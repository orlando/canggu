# Canggu

Docker containers stats visualizator, build using React + Electron. The
UI is heavily inspired in [Kitematic](https://github.com/docker/kitematic)

![gif](http://recordit.co/SXwGC9ToCU.gif)

## Development

```js
npm install
npm run compile
npm start
```

## Package

```js
npm run package
```
